### 2. Small Health GPT

Small Health GPT is a full-stack machine learning project that builds a small GPT-style model for health information.

The project is inspired by Andrej Karpathy’s transformer lectures and focuses on understanding how transformer-based language models work from the ground up.

This project includes a full pipeline:

- Dataset preparation
- Tokenization
- Transformer model training
- Text generation
- Backend API
- Frontend interface

#### Project Goal

The goal is to build a small local GPT-style assistant that can answer basic health-related questions using a custom-trained transformer model.

This is not meant to replace medical advice. It is an educational project for learning how language models are built, trained, and connected to a working application.

#### Main Ideas

- Build a small transformer model
- Train the model on health-related question-answer data
- Create a backend that loads the trained model locally
- Build a simple frontend where users can ask questions
- Connect the frontend, backend, and model into one working system

#### Skills Practiced

- Transformer architecture
- Language modeling
- Tokenization
- PyTorch
- Model training
- Text generation
- Backend development
- Frontend development
- Full-stack ML deployment
